Framework Capabilities Working Group

Frameworks and their features

The OpenVT project aims to provide a text-based schema that the community can utilize to both describe features of each modeling framework and those of individual models to ease cross framework descriptions. This schema currently defines objects and processes in order to create a bigraph and json describing the model in an easily digestible manner.

This is an ongoing project, and we are actively searching for contributors, a model prototype schema is accessible on GitHub.
